IsNumberEven. C++ Tutorials. Microsoft Visual Studio IDE. Simple Input/Output. Algorithms. Console application. 


A console application in the C++ programming language. 
The program does the following:
  1. Prompts input a natural number.
  2. Calculates a modulo to 2 of the number
  3. if the modulo is equal 0 the number is even or odd in other case
Developed in the Microsoft Visual Studio integrated environment.
